#单元测试框架
```
npm install -g karma 
npm install -g karma-jasmine karma-chrome-launcher
npm install -g karma-cli
初始化配置文件
karma init my.conf.js
开启
karma start my.conf.js
karma start my.conf.js --log-level debug --single-run
```
