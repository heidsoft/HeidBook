#null 判断
```
typeof null // "object"
typeof undefined //"undefined"
null == undefined //true
null === undefined //false
null !== undefined //true

```


#空字符判断
```
'' =='0' //false
'' ==='0' //false
0 == ''  // true
0 === '' //false
0=='0' //true
0==='0' //false

```     

#false判断
```
typeof false //"boolean"
false =='false' //false
false ==='false' //false
false ==='0' //false
false =='0' //true
```
